## 查看当前所有连接的详细资料:

```shell
mysqladmin -uroot -proot processlist
```

## 只查看当前连接数(Threads就是连接数.

```shell
mysqladmin -uroot -proot status
````

## mysql最大连接数：

```shell
cat my.cnf
  max_connections=100(默认为100)。
```
